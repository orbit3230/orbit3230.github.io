## 상관분석과 회귀분석
### 1. 상관분석  
- 임의의 확률변수 `X`와 `Y`가 주어질 때, `X`와 `Y`의 **공분산**(covariance) `Cov(X, Y)`를 다음과 같이 정의한다. (단, `μ`<sub>`X`</sub>, `μ`<sub>`Y`</sub>는 각각 `X`의 기댓값 `E[X]`, `Y`의 기댓값 `E[Y]`를 의미)  
`Cov(X, Y)` = `E[(X - μ`<sub>`X`</sub>`) (Y - μ`<sub>`Y`</sub>`)]` = `E(XY)` - `μ`<sub>`X`</sub>`μ`<sub>`Y`</sub>  
  - 두 개의 확률변수의 선형 관계를 나타내는 값
  - 같은 방향이면 양수, 반대 방향이면 음수, `0`에 가까우면 선형 관계가 약함을 의미  

<br>

- 공분산의 성질
  - (1) `Cov(X, X)` = `V(X)` = `σ`<sup>`2`</sup>
  - (2) `Cov(X, a)` = `0`
  - (3) `Cov(X, Y)` = `Cov(Y, X)`
  - (4) `Cov(aX, bY)` = `ab Cov(X, Y)`  
  - (5) `Cov(X + a, Y + b)` = `Cov(X, Y)`  
  - (6) `Cov(aX + b, cY + d)` = `ac Cov(X, Y)`
  - (7) `Cov(aX + bY, cW + dV)` = `ac Cov(X, W)` + `ad Cov(X, V)` + `bc Cov(Y, W)` + `bd Cov(Y, V)`
  - (8) 일반적으로 확률변수 `X`<sub>`i`</sub>(`i` = `1`, `2`, ..., `n`)와 `Y`<sub>`j`</sub>(`j` = `1`, `2`, ..., `m`)에 대하여 `X` = `a`<sub>`1`</sub>`X`<sub>`1`</sub> + `a`<sub>`2`</sub>`X`<sub>`2`</sub> + ... + `a`<sub>`n`</sub>`X`<sub>`n`</sub>, `Y` = `b`<sub>`1`</sub>`Y`<sub>`1`</sub> + `b`<sub>`2`</sub>`Y`<sub>`2`</sub> + ... + `b`<sub>`m`</sub>`Y`<sub>`m`이면 다음이 성립  
  ![covariance_property][def]  

<br>

- 공분산과 분산의 관계 : 확률변수 `X`, `Y`와 실수 `a`, `b`에 대해 다음이 성립한다.
  - (1) `V(aX + bY)` = `a`<sup>`2`</sup>`V(X)` + `b`<sup>`2`</sup>`V(Y)` + `2ab Cov(X, Y)`
  - (2) 만약 `X`와 `Y`가 독립이라면, `V(aX + bY)` = `a`<sup>`2`</sup>`V(X)` + `b`<sup>`2`</sup>`V(Y)`  

<br>

- 확률변수 `X`, `Y`가 주어질 때, `X`와 `Y`의 **상관계수**(correlation coefficient)를 다음과 같이 정의하고 `ρ`<sub>`XY`</sub> 또는 `Corr(X, Y)`로 표기한다. (단, `σ`<sub>`X`</sub>, `σ`<sub>`Y`</sub>는 각각 `X`, `Y`의 표준편차)  
![correlation_coefficient][def2]  

<br>

- 상관계수의 성질
  - (1) `Corr(X, X)` = `1`, `Corr(-X, X)` = `-1`  
  - (2) `Corr(X, Y)` = `Corr(Y, X)`
  - (3) `-1` ≤ `Corr(X, Y)` ≤ `1`
  - (4) `Corr(aX + b, cY + d)` = `Corr(X, Y)`, `ac` > `0`인 경우
  - (5) `Corr(aX + b, cY + d)` = `-Corr(X, Y)`, `ac` < `0`인 경우
  - (6) 만약 `X`와 `Y`가 독립이라면, `Corr(X, Y)` = `0`
