## Procedures
- Procedures의 메커니즘
  - Passing control
    - procedure code의 시작점으로 control이 넘어간다.
    - return point에서 다시 돌아간다.  
  - Passing data
    - Procedure arguments
    - Return value
  - Memory management
    - procedure 실행 도중 할당 (local variables)
    - return에서 해제(deallocation)
