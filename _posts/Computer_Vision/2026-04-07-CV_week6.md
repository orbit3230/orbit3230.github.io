<br>

- 소벨 에지 검출(`Sobel()` 함수 사용) 하기  

```python
import cv2 as cv

img = cv.imread('soccer.jpg')
gray = cv.cvtColor(img, cv.COLOR_BGR2GRAY)

grad_x = cv.Sobel(gray, cv.CV_32F, 1, 0, ksize=3)
grad_y = cv.Sobel(gray, cv.CV_32F, 0, 1, ksize=3)

sobel_x = cv.convertScaleAbs(grad_x)
sobel_y = cv.convertScaleAbs(grad_y)

edge_strength = cv.addWeighted(sobel_x, 0.5, sobel_y, 0.5, 0)

cv.imshow('Original', gray)
cv.imshow('sobelx', sobel_x)
cv.imshow('sobely', sobel_y)
cv.imshow('edge strength', edge_strength)

cv.waitKey()
cv.destroyAllWindows()
```

### 3. 직선 검출
- 에지를 명시적으로 연결하여 경계선을 찾고 직선으로 변환  
  - 이후 처리 단계인 물체 표현이나 인식에 유리  

<br>

#### [1] 경계선 찾기
- 8-연결된 에지 화소를 연결해 경계선(contour) 구성  
![contour_detection][def10]  

<br>

- 에지 맵에서 경계선 찾기  

```python
import cv2 as cv
import numpy as np

img = cv.imread('soccer.jpg')
gray = cv.cvtColor(img, cv.COLOR_BGR2GRAY)
canny = cv.Canny(gray, 100, 200)

contour, hierarchy = cv.findContours(canny, cv.RETR_LIST, cv.CHAIN_APPROX_SIMPLE)

lcontour = []
for i in range(len(contour)):
    if contour[i].shape[0] > 100:
        lcontour.append(contour[i])

cv.drawContours(img, lcontour, -1, (0, 255, 0), 3)

cv.imshow('Original with contours', img)
cv.imshow('Canny', canny)

cv.waitKey()
cv.destroyAllWindows()
```

#### [2] 허프 변환
- 허프 변환은 끊긴 에지를 모아 직선 또는 원 등을 검출
- 직선 검출의 원리
  - 각각의 점 `(y`<sub>`i`</sub>, `x`<sub>`i`</sub>`)`에 대해 `(b, a)` 공간에 직선 `b = -ax`<sub>`i`</sub> + `y`<sub>`i`</sub>`를 그림  
  - `(b, a)` 공간에서 직선이 만나는 점을 절편과 기울기로 취함. 만나는 점은 투표로 알아냄  
  ![hough_transform][def16]  

<br>

- 구현
  - `(b, a)` 공간을 이산화하고 누적 배열을 만들어 투표를 기록
  - 직선은 자신이 지나는 칸에 `1` 만큼 씩 투표
  - 다수 표를 얻은 점을 결정할 때 비최대 억제 적용하여 지역 최대점을 찾음  
  ![hough_transform_implementation][def18]  

  - 극좌표에서 정의된 직선의 방정식 사용 (기울기가 무한대인 경우 대처)  
  `x sin(θ)` + `y cos(θ)` = `ρ`  

<br>

- 원 검출을 위한 허프 변환은 3차원 누적 배열 사용  
`(x - a)`<sup>`2`</sup> + `(y - b)`<sup>`2`</sup> = `r`<sup>`2`</sup>  

<br>

- 허프 변환을 이용해 사과 검출하기  

```python
import cv2 as cv

img = cv.imread('apples.jpg')
gray = cv.cvtColor(img, cv.COLOR_BGR2GRAY)

apples = cv.HoughCircles(gray, cv.HOUGH_GRADIENT, 1, 200, param1=150, param2=20, minRadius=50, maxRadius=120)

for i in apples[0] :
  cv.circle(img, (int(i[0]), int(i[1])), int(i[2]), (255, 0, 0), 2)

cv.imshow('Apple detection', img)

cv.waitKey()
cv.destroyAllWindows()
```

#### [2] RANSAC
- 허프 변환과 최소평균제곱오차(LMSE; Least Mean Square Error)는,  
  - 강인하지 않은 추정 기법이다.
  - 아웃라이어를 걸러내는 기능이 전혀 없다. (모든 점에 같은 기회 부여)  
  ![hough_and_lmse][def19]  

<br>

- RANSAC(Random Sample Consensus)은 중앙값을 이용한 강인한 추정 기법.
  - RANSAC은 구체적인 문제에 두루 활용할 수 있는 메타 알고리즘
  - 난수를 생성하여 추정하는 일을 충분히 많이 반복하고 가장 신뢰할 수 있는 값 선택
