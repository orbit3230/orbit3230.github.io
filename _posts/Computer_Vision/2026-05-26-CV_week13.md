## 다층 퍼셉트론
### 1. 신경망 기초
- 신경망에는 아주 다양한 모델이 존재
  - 전방 신경망(feedforward)과 순환 신경망(recurrent)  
  - 얕은 신경망과 깊은 신경망
  - 결정론 신경망(deterministic)과 스토캐스틱 신경망(stochastic)  
  ![kinds_of_neural_networks][def]  

<br>

### 2. 퍼셉트론(Perceptron)  
#### [1] 구조
- 퍼셉트론 구조
  - 입력층과 출력층
    - 입력층은 연산을 하지 않으므로 퍼셉트론은 단일층 구조
  - 입력층의 `i`번째 노드는 특징 벡터 `x` = `(x`<sub>`1`</sub>`, x`<sub>`2`</sub>`, ..., x`<sub>`d`</sub>`)`<sup>`T`</sup>의 `i`번째 요소 `x`<sub>`i`</sub>를 나타냄
  - 항상 `1`이 입력되는 노드가 존재하는데, 이를 바이어스 노드라고 함
  - 출력층은 한 개의 노드
  - `i`번째 입력층 노드와 출력층을 연력하는 엣지(edge)는 가중치 `w`<sub>`i`</sub>를 가짐  
  ![perceptron][def2]  

<br>

#### [2] 동작  
- 퍼셉트론의 동작
  - 해당하는 특징값과 가중치를 곱한 결과를 모두 더하여 `s`를 구하고
  - **활성 함수(activation function) `τ`**를 적용
  - 활성함수 `τ`로 계단 함수(step function)를 사용하므로 최종 출력 `y`는 `+1` 또는 `-1`  
  ![perceptron_operation][def3]  

<br>

- 동작(학습이 끝난 상태)  
![perceptron_operation_after_learning][def4]  

<br>

- 행렬 표기  
`s = w`<sup>`T`</sup>`x + w`<sub>`0`</sub>  
  - 여기서 `x = (x`<sub>`1`</sub>`, x`<sub>`2`</sub>`, ..., x`<sub>`d`</sub>`)`<sup>`T`</sup>, `w = (w`<sub>`1`</sub>`, w`<sub>`2`</sub>`, ..., w`<sub>`d`</sub>`)`<sup>`T`</sup>

- 바이어스 항을 벡터에 추가하면,  
`s = w`<sup>`T`</sup>`x`
  - 여기서 `x = (1, x`<sub>`1`</sub>`, x`<sub>`2`</sub>`, ..., x`<sub>`d`</sub>`)`<sup>`T`</sup>, `w = (w`<sub>`0`</sub>`, w`<sub>`1`</sub>`, w`<sub>`2`</sub>`, ..., w`<sub>`d`</sub>`)`<sup>`T`</sup>  

- 퍼셉트론의 동작을 식으로 표현  
`y = τ(w`<sup>`T`</sup>`x)`  

<br>

- 위 그림을 기하학적으로 설명하면,
  - 결정 직선 `d(x) = d(x`<sub>`1`</sub>`, x`<sub>`2`</sub>`) = w`<sub>`1`</sub>`x`<sub>`1`</sub>` + w`<sub>`2`</sub>`x`<sub>`2`</sub>` + w`<sub>`0`</sub>` = 0`  
    - `w`<sub>`1`</sub>과 `w`<sub>`2`</sub>는 결정 직선의 기울기를 결정  
    - `w`<sub>`0`</sub>는 결정 직선의 절편
    - 전체 공간을 `+1`과 `-1`의 두 부분공간으로 분할하는 **선형 분류기**(linear classifier)  
  ![perceptron_decision_boundary][def5]  

<br>

#### [3] 학습
- 지금까지는 학습을 마친 퍼셉트론으로 동작을 설명

- 어떻게 학습시킬 것인가?  
![perceptron_learning][def6]  

<br>

- 목적합수(objective function, loss function) 설계  
  - 매개변수 `w = (w`<sub>`0`</sub>`, w`<sub>`1`</sub>`, w`<sub>`2`</sub>`, ..., w`<sub>`d`</sub>`)`<sup>`T`</sup>, 매개변수 집합 `Θ = {w}`  
  - 목적함수 : `J(Θ)` 또는 `J(w)`  
  ![perceptron_objective_function][def7]  
  - 목적함수의 조건
    - `J(w) ≥ 0`  
    - `w`가 최적이면, 즉 모든 샘플을 맞히면 `J(w) = 0`  
    - 틀리는 샘플이 많은 `w`일수록 `J(w)`는 큰 값을 가진다.  

  - 아래 식은 3가지 조건을 만족  
  ![perceptron_objective_function_formula][def8]  
    - `Y` : `w`가 틀리는 샘플의 집합
    - `y`<sub>`k`</sub> : `x`<sub>`k`</sub>의 부류 `1` 또는 `-1`, Ground Truth  

    <br>

- 그래디언트 계산
  - 위 식을 편미분  
  ![perceptron_gradient][def9]  

  - 위 편미분 결과를 대입하여 델타 규칙(delta rule) 도출  
  ![perceptron_delta_rule][def10]  

<br>

- 델타 규칙 식을 이용한 학습 알고리즘  
![perceptron_learning_algorithm][def11]  
  - 훈련집합의 샘플을 모두 맞추는, 즉 `Y = ∅`이 될 때까지 epoch 반복  

- 퍼셉트론 학습 알고리즘의 스토캐스틱 버전  
![perceptron_stochastic_learning_algorithm][def12]  
  - 샘플 순서를 섞음
  - 틀린 샘플이 발생하면 즉시 갱신  

<br>

- 학습 예제

![perceptron_learning_example_2][def14]

### 3. 다층 퍼셉트론
- 퍼셉트론은 선형 분류기라는 한계. 선형 분리 불가능한 상황에서는 일정한 양의 오류  
![perceptron_non_linear][def16]  

<br>

#### [1] 특징 공간 변환
- 퍼셉트론 2개를 사용한 XOR 문제 해결  
![xor_problem][def17]  
  - 퍼셉트론 ①과 퍼셉트론 ②가 모두 `+1`이면 ● 부류이고 그렇지 않으면 □ 부류  

<br>

- 퍼셉트론 2개를 병렬로 결합  
![parallel_perceptron][def18]
  - 원래 공간 `x = (x`<sub>`1`</sub>`, x`<sub>`2`</sub>`)`<sup>`T`</sup>를 새로운 특징 공간  `z = (z`<sub>`1`</sub>`, z`<sub>`2`</sub>`)`<sup>`T`</sup>로 변환  
  - 새로운 특징 공간 `z`에서는 선형 분리 가능  
  - `z`<sub>`k`</sub>`= {1, -1}`, 따라서 점으로 매핑  

<br>

- 퍼셉트론 2개를 순차 결합  
![sequential_perceptron][def19]  
  - 새로운 특징 공간 `z`에서 선형 분리를 수행하는 퍼셉트론 ③을 순차 결합  

<br>

- 다층 퍼셉트론의 용량  
![multilayer_perceptron_capacity][def20]  
  - 위 그림처럼 ①, ②, ③의 결정직선을 결합한 퍼셉트론을 가정했을 때
  - 활성함수 `τ`가 계단 함수이므로, 영역을 점으로 변환, `z`<sub>`k`</sub>`= {1, -1}`  
  - 2차원 공간을 7개 영역으로 나누고 각 영역을 3차원 점으로 변환  

- 즉, 일반화하면 `p`개 퍼셉트론을 결합하면 `p`차원 공간으로 변환, `1 + (1 + 2 + ... + p)`개의 영역으로 분할  

<br>  

#### [2] 활성함수
- 딱딱한 공간 분할과 부드러운 공간 분할  
![activation_function][def21]  
  - 계단 함수는 딱딱한 의사 결정 (영역을 점으로 변환)
  - 나머지 활성함수는 부드러운 의사결정(영역을 영역으로 변환)  

<br>

- 신경망이 사용하는 다양한 활성함수  
![activation_functions][def22]  
  - 로지스틱 시그모이드와 하이퍼볼릭 탄젠트는 `a`가 커질수록 계단함수에 가까워짐
  - 모두 1차 도함수 계산이 빠름 (특히 ReLU)  
  - 퍼셉트론은 계단함수, 다층 퍼셉트론은 로지스틱 시그모이드와 하이퍼볼릭 탄젠트
  - 딥러닝은 ReLU  
  
<br>

- 시그모이드 미분  
![sigmoid_derivative][def23]  

- Gradiant Vanishing  
![gradient_vanishing][def24]  

<br>

#### [3] 구조  
![multilayer_perceptron_structure][def25]  
- 그림 (a)는 입력층-은닉층-출력층의 2층 구조
  - `d+1`개의 입력노드, `c`개의 출력노드
  - `p`개의 은닉노드 : `p`는 사용자가 지정하는 하이퍼 파라미터로, 너무 크면 과잉적합, 너무 작으면 과소적합  

- 그림 (b)는 입력층-은닉층-은닉층-출력층의 3층 구조  

<br>

- 다층 퍼셉트론의 매개변수(가중치)  
![multilayer_perceptron_parameters][def26]  
  - 입력층-은닉층을 연결하는 `U`<sup>`1`</sup> (`u`<sup>`1`</sup><sub>`ji`</sub> : 입력층의 `i`번째 노드 - 은닉층의 `j`번째 노드 연결)  
  - 은닉층-출력층을 연결하는 `U`<sup>`2`</sup> (`u`<sup>`2`</sup><sub>`kj`</sub> : 은닉층의 `j`번째 노드 - 출력층의 `k`번째 노드 연결)  

- 다층 퍼셉트론은 분산처리와 병렬처리를 수행하므로 병렬 분산 구조  

<br>

#### [4] 동작
- 특징 벡터 `x`를 출력 벡터 `o`로 매핑하는 함수로 간주  
![multilayer_perceptron_mapping][def27]  

<br>

- 노드가 수행하는 연산을 구체적으로 쓰면,  
![multilayer_perceptron_node_operation][def28]  

- 다층 퍼셉트론의 동작을 행렬로 표기하면,  
`o = τ(U`<sup>`2`</sup>`τ(U`<sup>`1`</sup>`x))`  

<br>

- 은닉층은 특징 추출기(feature extractor)  
![multilayer_perceptron_feature_extractor][def29]  
  - 은닉층은 특징 벡터를 분류에 더 유리한 새로운 특징 공간(feature space)으로 변환  
  - **특징 학습**(feature learning)  
  - 딥러닝은 더 많은 단계를 거쳐 특징 학습을 수행  
  - 딥러닝 : 여러 단계의 은닉층이 자동으로 계층적인 특징 추출
