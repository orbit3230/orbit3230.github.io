## 영상 처리
### 1. 디지털 영상 기초
- 영상 처리 : 특정 목적을 달성하기 위해 원래 영상을 개선된 새로운 영상으로 변환하는 작업  
![image_processing_example][def]  

<br>

#### [1] 영상 획득과 표현
- 핀홀 카메라 모델
  - 영상 획득 과정은 매우 복잡
  - 핀홀 카메라 모델은 핵심을 설명  
  ![pin_hole_camera_model][def2]  

<br>

- 디지털 변환
  - `M` * `N` 영상으로 샘플링 (sampling)
  - `L` 단계로 양자화 (quantization)  
  ![digital_conversion][def3]  

<br>

- 영상 좌표계
  - 왼쪽 위 구석이 원점
  - `(y, x)` 표기  
  ![image_coordinates][def4]  

  - 다만 함수에 따라 `(x, y)` 표기도 존재  
  `cv.line(img, (x1, y1), (x2, y2), ...)`  

  - OpenCV는 `numpy.ndarray`로 영상 표현  
    - `numpy.ndarray`가 지원하는 다양한 함수를 사용할 수 있다는 큰 장점
    - e.g., `min`, `max`, `argmin`, `argmax`, `mean`, `sort`, `reshape`, `transpose`, `flatten` 등

```python
import cv2 as cv
import sys

img = cv.imread('soccer.jpg')

    sys.exit("파일을 찾을 수 없습니다.")

cv.imshow("original_RGB", img)

cv.imshow("R channel", img[:, :, 2])
cv.imshow("G channel", img[:, :, 1])
cv.imshow("B channel", img[:, :, 0])

cv.waitKey(0)
cv.destroyAllWindows()
```

### 2. 이진 영상
- 이진 영상
  - 화소가 `0`(흑) 또는 `1`(백)인 영상  
  - 1비트면 저장할 수 있는데, 편의상 1바이트 사용하는 경우 많음
  - 엣지 검출 결과를 표시하거나 물체와 배경을 구분하여 표시하는 응용 등에 사용

<br>

#### [1] 이진화 (binarization)  
- 알고리즘
  - 임계값 `T`보다 큰 화소는 `1`, 그렇지 않은 화소는 `0`으로 바꿈. 임계값 결정이 중요  
  ![binary_image_algorithm][def9]  

  - 히스토그램에서 계곡 부분을 임계값으로 결정하는 방법  
  ![thresholding_histogram][def10]  

  - 실제 영상에서는 계곡이 아주 많이 나타나서 구현이 쉽지 않다.  

  ```python
  import cv2 as cv
  import matplotlib.pyplot as plt
  
  img = cv.imread('soccer.jpg')
  h = cv.calcHist([img], [2], None, [256], [0, 256])
  plt.plot(h, color='r', linewidth=1)  
  ```

  ![code_output_histogram][def11]  

<br>

#### [2] 오츄 알고리즘 (Otsu's algorithm)  
- 이진화를 최적화 문제로 바라봄. 최적값 `t̂`을 임계값 `T`로 이용  
![otsu_algorithm][def13]  

- 목적 함수 `J(t)`는 임계값 `t`의 좋은 정도를 측정함 (작을수록 좋음)  
  - `t`로 이진화했을 때 `0`이 되는 화소들의 분산 `(v`<sub>`0`</sub>`(t))`과 `1`이되는 화소들의 분산 `(v`<sub>`1`</sub>`(t))`의 가중치 `(n`<sub>`0`</sub>`(t))`와 `(n`<sub>`1`</sub>`(t))`의 합을 `J`로 사용  
  ![otsu_objective_function][def12]  

<br>

- 오츄 알고리즘으로 이진화하기  

```python
import cv2 as cv
import sys

img = cv.imread('soccer.jpg')

t, bin_img = cv.threshold(img[:, :, 2], 0, 255, cv.THRESH_BINARY + cv.THRESH_OTSU)
print("오츄 알고리즘이 찾은 최적 임곗값 = ", t)

cv.imshow("R channel", img[:, :, 2])
cv.imshow("R channel binarization", bin_img)

cv.waitKey(0)
cv.destroyAllWindows()
```
