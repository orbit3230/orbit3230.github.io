# 중첩 문장
## 1. 중첩 if 문장 (nested if 문장)
```java
int grade = input.nextInt();
if(grade >= 70) {  // 이 괄호는 생략 가능하다. if ~ else 는 한 문장!
    if(grade >= 90) {
        System.out.print("Distinct");
    }
    else {
        System.out.print("Pass");
    }
}
else {
    System.out.print("Fail");
}
```
- <sup> `else`는 가까이있는 `if`와 짝을 이룬다.  

- 프로그램 예  
```java
if(testscore >= 90) {  // testscore 변수는 int 타입으로 선언되어 값이 들어있음
    grade = 'A';  // grade 변수는 char 타입으로 이미 선언된 상태로 가정
}
else if(testscore >= 80) {
    grade = 'B';
}
else if(testscore >= 70) {
    grade = 'C';
}
else if(testscore >= 60) {
    grade = 'D';
}
else {
    grade = 'F';
}
```
- 위 코드는 아래와 완전히 같다.
```java
if(testscore >= 90) {  // testscore 변수는 int 타입으로 선언되어 값이 들어있음
    grade = 'A';  // grade 변수는 char 타입으로 이미 선언된 상태로 가정
}
else { 
    if(testscore >= 80) {
        grade = 'B';
    }
    else {
        if(testscore >= 70) {
            grade = 'C';
        }
        else {
            if(testscore >= 60) {
                grade = 'D';
            }
            else {
                grade = 'F';
            }
        }
    }
}
```

## 중첩 for 문장 (nested for 문장)
```java
for(int i = 0 ; i < 2 ; i++) {
    for(int j = 0 ; j < 3 ; j++) {
        System.out.print("(" + i + "," + j + ")" + "\t");
    }
    System.out.println();
}
```
- 출력 결과는 다음과 같다.
```
(0,0)   (0,1)   (0,2)
(1,0)   (1,1)   (1,2)
(2,0)   (2,1)   (2,2)
```
